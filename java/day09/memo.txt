상속성
상속이해
	클래스간의 상속이란
		클래스간에는 부모-자식의 상속관계를 설정할 수있다
		부모 클래스 a를 자식 클래스 b가 상속받을 때
		b는 a의 모든 멤버변수와 메서드를 자신의 것으로 상속받게 괸다
		(단 private으로 설정된 기능은 상속되지 않는다.)
		b는 a의 public prtected 기능들은 직접적으로 코딩하지 않더라고
		자신의 것으로 사용할 수 있게 된다

상속 정의 방법
	extends 키워드를 사용하여 부모클래스의 이름을 명시한다.
	
	public class 자식클래스 extends 부모클래스{}
